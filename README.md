# Project 4: Form Design & Validation


## 🌟 Overview

A modern, fully responsive form with real-time validation, beautiful UI animations, and a live data display panel. Built with pure HTML, CSS, and JavaScript as part of the **Decode Labs Internship** program.

> 🏢 **Internship Project**: This project was developed during my internship at [Decode Labs](https://decodelabs.com)

## ✨ Features

### 🎨 Visual Features
- **Gradient backgrounds** with animated effects
- **Glass-morphism design** with backdrop blur
- **Smooth animations** & transitions
- **Responsive layout** for all devices
- **Interactive hover & focus states**

### ✅ Validation Features
- **Real-time validation** on blur & input
- **Visual feedback** with success/error states
- **Character counters** with warning indicators
- **Progress tracking** for form completion
- **Status badges** showing form state

### 📊 Data Management
- **Live data panel** displaying submitted info
- **Clear data** functionality
- **Submission count** tracking
- **Console logging** for debugging

### 🌍 Rich Select Options
- **190+ countries** with flag emojis
- **8 subject categories** with icons
- **Styled select dropdowns**

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Animations, gradients, flexbox
- **Vanilla JavaScript** - DOM manipulation, validation logic
- **Google Fonts** - Clean typography
